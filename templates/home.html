California Housing Dataset

Data Set Characteristics

Number of Instances: 20,640

Number of Attributes: 8 numeric predictive attributes + 1 target variable

Missing Attribute Values: None

Attribute Information

  • MedInc – Median income in block group
  • HouseAge – Median house age in block group
  • AveRooms – Average number of rooms per household
  • AveBedrms – Average number of bedrooms per household
  • Population – Block group population
  • AveOccup – Average number of household members
  • Latitude – Block group latitude
  • Longitude – Block group longitude

Target Variable

The target variable is the median house value for California districts, expressed in hundreds of thousands of dollars ($100,000 units).

Dataset Description

This dataset was derived from the 1990 U.S. Census, using one row per census block group. A block group is the smallest geographical unit for which the U.S. Census Bureau publishes sample data (typically between 600 and 3,000 people).

A household is defined as a group of people residing within a home. Since the average number of rooms and bedrooms are provided per household, these columns may contain large values in block groups with few households and many empty houses (e.g., vacation resorts).

Model Selection & Performance

The selected model for this application is LightGBM. Compared to AdaBoost, Linear Regression, and XGBoost, LightGBM achieved the best performance with an R² score of 0.866 on the test dataset.
